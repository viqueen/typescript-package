## typescript-package

Template for building node packages using Typescript

### environment

- **[nvm](https://github.com/nvm-sh/nvm)** to manage node versions.

```bash
brew install nvm
```

- install node version

```bash
nvm install
```

### house-keeping

- install dependencies

```bash
npm ci
```

- build it

```bash
npm run build
```

- format it

```bash
npm run  format
```

- lint it

```bash
npm run lint
npm run lint --fix
```

- test it

```bash
npm test
```
